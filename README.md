# Oykus

A project.

## Custom Frontend Plugins
- Translations (i18n)
- Seamless routing (router)
- Stores (store)

## Installation

### Backend (django)
```
py -m venv .venv
```
```
source .venv/Scripts/activate (deactivate)
```
```
py -m pip install --upgrade pip
```
```
py -m pip install -r requirements.txt
```
```
py manage.py migrate
```
```
py manage.py runserver
```

### Frontend (react)
```
npm install
```
```
npm run start
```
