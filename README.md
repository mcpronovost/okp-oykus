# Oykus

A project.

## Installation

### Backend
```
py -m venv .venv
```
```
source .venv/Scripts/activate (deactivate)
```
```
py -m pip install --upgrade pip
```
```
py -m pip install -r requirements.txt
```
```
py manage.py migrate
```
```
py manage.py runserver
```

### Frontend
```
npm install
```
```
npm run start
```
